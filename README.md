# Backup Automation

Diese einfache bash lädt Dateien von https://getsamplefiles.com/sample-document-files/word herunter, komprimiert diese und speichert sie als Backup.

Ein Beispiel für die zeitgesteuerte Ausführung mittels cron befindet sich unter:
- cron/backup.cron
Dies müsste im cron editor dann entsprechend konfiguriert werden.
